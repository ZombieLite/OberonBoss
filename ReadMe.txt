Макросы босса
=====================
#define MAPCHOOSER		// Включите это, если используете API плагин мапчузера.
#define PLAYER_HP			// Если включено, здоровье босса будет умножено на кол-во игроков.
#define NEW_SEARCH		// Если включено, то босс будет преследовать ближайшего игрока
#define BOMB_CUSTOM		// Если включено, будут использоваться новые типы бомб ( custom bomb )
#define RANDOM_ABILITY		// Если включено, босс будет использовать способности в разноброс
#define SUPPORT_ZM		// Поддержка [API] ZombieSystem
#define BLOOD_DECAL		// Включить/отключить декали следов крови ( желтые от босса )
#define HEALTHBAR			// Позволяет изменить тип бара со здоровьем ( Как у ангры или спрайт )
#define SPRMIRROR			// Если вы используете кастомный спрайт-бар со здоровьем
#define SPRSIZE			// Размер спрайта

Все остальные настройки в файле "configs/zl/zl_oberonboss.ini"
